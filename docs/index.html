Mouse Highlighter

An extension for Google Chrome that highlights the mouse cursor so that it's a little easier to see when screen sharing.

Installation

Open the Mouse Highlighter extension from the Chrome Web Store by clicking Add to Chrome.

Source Code

The open source version can be cloned from the mouse-highlighter project on GitHub and installed from there.

  • Clone the repo
  • Open Extensions (Menu, More Tools, Extensions) in Google Chrome
  • Check the "Developer mode" checkbox
  • Click "Load unpacked extension"
  • Point it to the directory where you cloned the repo
